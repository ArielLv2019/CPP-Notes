## Lambda 表达式
```
Lambda 表达式就是用于创建匿名函数的。
为什么说 lambda 表达式如此激动人心呢？举一个例子。标准 C++ 库中有一个常用算法的库，其中提供了很多算法函数，比如 sort() 和 find()。
这些函数通常需要提供一个“谓词函数 predicate function”。所谓谓词函数，就是进行一个操作用的临时函数。比如 find() 需要一个谓词，用于查找元素满足的条件；
能够满足谓词函数的元素才会被查找出来。这样的谓词函数，使用临时的匿名函数，既可以减少函数数量，又会让代码变得清晰易读。
```
```cpp
[capture](parameters) -> return-type {
  body
}       
```
```cpp
void absSort(float* x, unsigned N){
  std::sort(x, x+N, [](float a, float b){
    return std::abs(a) < std::abs(b);
    });
}
